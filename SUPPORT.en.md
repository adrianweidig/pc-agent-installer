# Support

🌐 Languages: [Deutsch](SUPPORT.md) | [English](SUPPORT.en.md)

## Where Can I Get Help?

For questions and bug reports about the public template, use GitHub Issues:

- bug report: `.github/ISSUE_TEMPLATE/bug_report.yml`
- improvement proposal: `.github/ISSUE_TEMPLATE/feature_request.yml`
- documentation note: `.github/ISSUE_TEMPLATE/documentation.yml`

Describe the affected area, checks you ran, and the expected change as concretely as possible.

## What Does Not Belong in Public Issues?

Do not post:

- secrets, tokens, passwords, or private keys
- real host names or private local paths
- production kubeconfigs
- unredacted logs
- internal infrastructure details

Security-sensitive reports belong under `SECURITY.en.md`.

## Expected Support

This repository is a template, not a managed product service. Support is best effort through issues and pull requests. Operators remain responsible for their private operational repositories.
