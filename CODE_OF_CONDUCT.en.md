# Code of Conduct

🌐 Languages: [Deutsch](CODE_OF_CONDUCT.md) | [English](CODE_OF_CONDUCT.en.md)

## Expectation

This project should enable respectful, factual, and safe collaboration. Everyone should be able to contribute constructively, regardless of experience level, background, or work style.

## Expected Behavior

- communicate kindly and concretely
- focus technical criticism on the content
- state assumptions openly
- prefer small, reviewable contributions
- respect safety boundaries and privacy
- do not share host data or secrets publicly

## Unacceptable Behavior

- personal attacks, insults, or dismissive comments
- publishing private information without consent
- intentionally sharing secrets, tokens, or confidential host data
- repeatedly ignoring the public/private safety boundaries
- disruptive or intimidating behavior in issues, pull requests, or discussions

## Reporting

Report behavior that violates these rules through the GitHub channels available in the repository. For security-sensitive content, `SECURITY.en.md` also applies; confidential details do not belong in public issues.

## Enforcement

Maintainers may edit or hide comments, close issues, reject contributions, or exclude people from further participation when needed to protect the project and participants.
